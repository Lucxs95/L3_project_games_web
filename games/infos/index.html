Règles GAME 2:

Toutes les cartes sont étalées faces cachées. Le joueur retourne deux cartes.
Si c'est la même image qui apparaît sur les deux cartes le joueur garde son score.
Si les deux cartes ne vont pas ensemble, le joueur les replace face cachée et perd un point.

Celui qui a une bonne mémoire se souviendra de l'emplacement des cartes déjà retournées
et pourra les utiliser lorsqu'il en aura besoin pour compléter une paire.
